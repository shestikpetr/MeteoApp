# MeteoApp

Android-приложение для мониторинга данных метеостанций.

## Возможности

- Регистрация и авторизация пользователей
- Просмотр списка привязанных метеостанций
- Добавление и удаление станций
- Просмотр параметров станции в реальном времени
- Отображение станций на карте (OpenStreetMap)
- Просмотр статистики и истории измерений

## Стек

- **Kotlin** + **Jetpack Compose**
- **Material 3**
- **Retrofit** — работа с REST API
- **DataStore** — хранение токенов
- **Navigation Compose** — навигация
- **osmdroid** — карта OpenStreetMap
- **Google Play Services Location** — геолокация

## Сборка

1. Открыть проект в Android Studio.
2. В `local.properties` прописать адрес бэкенда:
   ```
   METEO_BASE_URL=https://your-backend.example.com/
   ```
   Значение также можно задать переменной окружения `METEO_BASE_URL`.
   Если ничего не задано, используется заглушка `https://example.com/`.
3. Запустить на устройстве/эмуляторе с Android 8.0+ (API 26).
